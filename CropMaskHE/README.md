These images are from DOI: [10.1038/s41588-024-01819-2](https://doi.org/10.1038/s41588-024-01819-2)

We leveraged the full resolution H&E images of human tissue sections used in Franzen et al., 2024  https://doi.org/10.1038/s41588-024-01819-2 from their data repository (BioStudies S-BSST1410): https://www.ebi.ac.uk/biostudies/studies/S-BSST1410. Their healthy control samples were labeled as A1. Their naming convention was to label IPF progression tissue blocks in increasing extent of fibrotic injury in the same donor as B1, B2, and B3. Then the image file names ended in B1, C1, and D1 for these levels of severity. The image files names are preserved here.

Details of the conversion to binary masks with ImageJ and csv files for NetLogo are described in the Initialization section of the manuscript. The file batch.txt is the script for automating the analysis for each original image.
